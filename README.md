# 🚢 Titanic Dashboard Analysis

## 📌 Project Overview

This project presents an interactive Power BI dashboard built using the Titanic dataset. The dashboard provides insights into passenger demographics, survival patterns, ticket classes, and other key factors that influenced survival during the Titanic disaster.

## 🎯 Objectives

* Analyze passenger survival rates
* Explore demographic distributions
* Compare survival by gender and passenger class
* Identify key trends and patterns through visualizations

## 📊 Dashboard Highlights

* Total Passengers Overview
* Survival vs Non-Survival Analysis
* Gender-wise Survival Distribution
* Passenger Class Analysis
* Age Group Insights
* Interactive Filters and Slicers

## 🛠️ Tools Used

* Power BI
* Data Cleaning & Transformation
* Data Visualization

## 📈 Key Insights

* Female passengers had a higher survival rate.
* First-class passengers were more likely to survive.
* Age and passenger class significantly influenced survival outcomes.
